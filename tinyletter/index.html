Mark Wunsch /tinyletter

This Week's Program is the name of my TinyLetter, published (as you would imagine) weekly. Every weekday I commit some code to a public repository, and the Friday of that week I send a summary of the commits and a report on project status. Publishing the newsletter is a forcing function to encourage a deliberate practice of writing code. The letters are committed to my website, and then published to TinyLetter through a git pre-push hook.
